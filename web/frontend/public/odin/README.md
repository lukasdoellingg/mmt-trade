# Odin WASM Runtime (MMT-Referenz)

Lege die von MMT geteilte **`odin.js`** hier ab als:

```
web/frontend/public/odin/odin.js
```

Dann ist sie unter `/odin/odin.js` erreichbar (Vite `public/`).

Die vollständige Datei enthält `WebGLInterface` + `webgl2` — unsere gekürzte Kopie wäre unbrauchbar.

Siehe `docs/MMT_ODIN_RUNTIME.md` und `docs/MMT_DEVTOOLS.md`.
